Angular CLI and Setup

Für Angular ist die CLI "ng" notwendig. Installation dieser App in der aktuellsten Version zu installieren : 

npm install -g @angular/cli
Offizielle Hilfe : https://angular.io/cli
ng new <name> Erstellt eine neue Angular - Anwendung im angegeben Verzeichnis.
ng (g)enerate <TYPE> <name>

Erstellt neue Dateien im aktuellen Projekt

TYPE= (c)omponent (s)ervice (d)irective

Nützliche Optionen:

-g => skip git
ng serve Startet den DevServer, und lässt die Anwendung auf http://localhost:4200
ng test / ng build Macht was der parameter verspricht. Siehe die Datei "angular.json" für eine genaue definition, bzw. "package.json" für den aufruf via npm
ng upgrade

Probiert ein Angular-Project auf die aktuellste Version zu bringen. Siehe auch https://update.angular.io/

Ich habe hier nicht viel erfolg gehabt -- häufig hat nur das Kopieren von src/ in ein komplett neues Projekt geholfen.
